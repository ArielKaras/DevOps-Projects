# Project-1: User-IP-Viewer

User-IP-Viewer is a web application built with React and Node.js that retrieves and displays the user's IP address.

## Prerequisites

- Node.js
- npm or yarn
- A code editor (e.g., Visual Studio Code)

## Installation

1. Clone the repository or download the project files.
2. Navigate to the `Project-1` directory in your terminal.
3. Run `npm install` or `yarn install` to install the required dependencies.
4. Start the Node.js server by running `node server.mjs` or `npm run start-server`.
5. In a separate terminal, navigate to the `Project-1/client` directory.
6. Run `npm install` or `yarn install` to install the required dependencies for the React application.
7. Start the React development server by running `npm start` or `yarn start`.

## Usage

1. Open your browser and go to `http://localhost:3000` to access the User-IP-Viewer web application.
2. The application will display your current IP address.
